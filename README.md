# Notes

Note taking app built with [React](https://reactjs.org/) + [wouter](https://github.com/molefrog/wouter) + [Tailwind CSS](https://tailwindcss.com/)

## Build setup

```bash
# install dependencies
pnpm install

# serve in dev mode
pnpm run dev

# build for production
pnpm run build

# serve in production mode
pnpm preview
```
